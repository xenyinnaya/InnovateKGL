# innovateKGL Hackathon Website

This is the source code for the innovateKGL online hackathon website.

## Getting Started

1.  **Open the project**: Double-click `open_project.bat` (requires VS Code).
2.  **Run locally**: Open `index.html` in your browser or use a live server.

## Features
-   Responsive Design
-   Registration Modal
-   Interactive Particle Background
